National Service

ü™ñ

Years Active: 2016 - 2018

In January 2016, I commenced my national service, initially met with apprehension due to the formidable reputation of military life, where recruits are subject to rigorous training under the guidance of sergeants and officers. Like most Singaporean males, my journey commenced with basic military training (BMT) in Gryphon company.


Given that I belonged to the cohort designated as 'obese,' my BMT extended over a span of 19 weeks, primarily dedicated to shedding excess weight. I successfully shed nearly 20 kilograms during this period. Beyond the enhancement of physical fitness, BMT instilled invaluable qualities of mental resilience and preparedness for the defense of Singapore, if the need ever arises. Memorable moments within this phase included live weapon firing and the exhilarating experience of throwing a live grenade, combining trepidation with a sense of thrill.

Following BMT, I was stationed at Seletar Camp, where my primary responsibilities revolved around the operation of heavy machinery, including bulldozers and excavators.

One of the proudest moments of my service was the privilege of participating in the National Day Parade (NDP) in 2017. In retrospect, my national service tenure was not only an enriching and transformative experience but also a vital chapter in my personal journey towards maturity." As the adage goes, I underwent a transformation from 'Ah Boy to Man'.

Programming Languages

Python

Python is a versatile and high-level programming language known for its readability and simplicity. It's widely used in web development, scientific computing, data analysis, and more. Python's extensive libraries and frameworks make it a go-to choice for various applications.

Java

Java is a robust, object-oriented programming language known for its platform independence. It's commonly used in developing Android apps, web applications, and enterprise-level software. Java's strong typing and vast ecosystem make it a reliable choice for large-scale projects.

JavaScript

JavaScript is a dynamic scripting language primarily used for front-end web development. It enables interactivity on websites, enhancing user experiences with features like form validation, animations, and dynamic content loading.

C#

C# is a versatile language developed by Microsoft, widely used for Windows application development, game development (in Unity), and web applications. Its strong typing and support for object-oriented programming make it suitable for a range of software development tasks.

Database Query Languages

SQL

Structured Query Language (SQL) is a domain-specific language used for managing and querying relational databases. SQL enables developers to interact with databases to retrieve, update, and manipulate data efficiently.

Web Development

HTML

Hypertext Markup Language (HTML) is the backbone of web content. It structures web pages by defining elements like headings, paragraphs, and links. HTML works hand in hand with CSS and JavaScript to create interactive and visually appealing websites.

CSS

Cascading Style Sheets (CSS) is used to control the presentation and layout of web pages. It defines styles like fonts, colors, spacing, and positioning, allowing developers to customize the appearance of HTML elements.

REST APIs

Representational State Transfer (REST) is an architectural style for designing networked applications. REST APIs enable communication between different software systems over HTTP. They're a fundamental part of modern web development.

Spring Boot

Spring Boot is a Java-based framework that simplifies the development of web applications and microservices. It provides tools and conventions to set up a production-ready application quickly.

Version Control

Git

Git is a distributed version control system used for tracking changes in source code during software development. It facilitates collaboration among developers, helping manage codebase versions and history.

GitHub

GitHub is a web-based platform that provides Git repository hosting, collaboration features, and tools for managing and tracking code changes. It's widely used for open-source and private project hosting.

Bitbucket

Bitbucket is a web-based platform for version control and collaborative software development. It offers Git and Mercurial repository hosting along with features for continuous integration.

Continuous Integration and Deployment

Jenkins

Jenkins is an automation server that streamlines the continuous integration and continuous delivery (CI/CD) process. It automates building, testing, and deploying software, increasing development efficiency.

Cloud Platforms

Microsoft Azure

Microsoft Azure is a cloud computing platform that offers a wide range of services, including infrastructure as a service (IaaS), platform as a service (PaaS), and software as a service (SaaS). It's used for hosting, managing, and scaling applications and services.

Pivotal Cloud Foundry

Pivotal Cloud Foundry is a cloud-native platform that simplifies application deployment and management. It supports multiple programming languages and provides tools for scaling and monitoring applications.

Code Quality and Security

SonarQube

SonarQube is an open-source platform for continuous inspection of code quality. It identifies code issues, bugs, and vulnerabilities, promoting codebase health and maintainability.

Fortify

Fortify is an application security testing tool that scans software code for security vulnerabilities. It helps organizations identify and fix potential security threats in their applications.

Nexus IQ

Nexus IQ is a component intelligence platform that identifies and manages open-source component risks in applications. It helps ensure that software projects use secure and compliant libraries.

Monitoring and Analytics

Grafana

Grafana is an open-source platform used for monitoring and observability. It helps visualize and analyze data from various sources, making it valuable for tracking system performance and making data-driven decisions.

Kibana

Kibana is part of the Elasticsearch ELK stack and is used for data exploration and visualization. It's particularly useful for log and event data analysis, making it a critical tool for monitoring and troubleshooting.

Project Management

JIRA

JIRA is a widely used project management and issue tracking tool developed by Atlassian. It aids in organizing tasks, tracking progress, and managing workflows, making it valuable for agile software development and project management.
